# Guessing Game 🎮

A simple number guessing game built using Unity and C#.

## About
In this game, the player has to guess a randomly generated number.  
The game gives hints like:

- Too High ⬆️
- Too Low ⬇️
- Correct ✅

This project was built to practice:
- C# scripting
- Unity UI system
- TextMeshPro
- Input handling
- Game logic

## Features
- Random number generation
- User input system
- Result display
- Simple UI
- Replay support

## Built With
- Unity
- C#
- TextMeshPro
